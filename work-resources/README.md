# Work Resources

This folder is for work-related files that will later be organized and linked from a webpage.

## Suggested organization

As files are added, they can be grouped into subfolders such as:

- `agreements/` — collective bargaining agreements, arbitration awards, and protective-condition documents
- `schedules/` — job schedules, crew assignments, and establishment-change notices
- `claims/` — claim references, examples, and supporting material
- `safety/` — FRA, OSHA, heat, equipment, and operating-safety resources
- `forms/` — reusable forms, templates, and handouts
- `reference/` — general work references that do not fit elsewhere

## File naming

Use clear, web-friendly names with lowercase letters and hyphens when practical.

Example: `beaumont-job-schedule-july-2026.pdf`

This README can later become the source list or index for the work-resources webpage.
